JAVA入门

一、环境配置

(1)下载环境

建议在官网下载https://www.java.com/

(2)安装

~
略
~

(3)配置环境

1.右键此电脑点击属性
2.点击左侧高级系统设置

3.点击环境变量

4.新建环境变量(注意标点)

JAVA_HOME(JAVA环境的安装路径)

ClassPath(.;%Java_Home%\bin;%Java_Home%\lib\dt.jar;%Java_Home%\lib\tools.jar)

Path(%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin)

5.测试
  1. win+R输入cmd，打开DOS窗口
  2. 输入java或javac，回车
  3. 一长串JAVA指令及用法，则配置成功
